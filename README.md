# AWS 3-Tier Web Application Project

## Overview
This project demonstrates a 3-tier web application hosted on AWS using:
- **Web Tier:** EC2 + ALB
- **App Tier:** EC2 instances in private subnets
- **Database Tier:** RDS (MySQL)
